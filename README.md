# 🎬 Movie Recommendation System

## 🚀 Overview

This project is a **Movie Recommendation System** built using collaborative filtering techniques and deployed as an interactive web application using Streamlit.

It provides real-time movie recommendations based on similarity between items, offering a user-friendly interface similar to modern streaming platforms.

---

## 🎯 Objective

To build and deploy a recommendation engine that suggests similar movies based on user selection using machine learning techniques.

---

## 🧠 Key Features

* 🎬 Movie-based recommendation system
* 🔍 Real-time recommendations
* 🖼️ Movie posters using TMDB API
* ⚡ Fast similarity-based predictions
* 🌐 Deployed web application

---

## 🛠️ Tech Stack

* **Python**
* **Pandas, NumPy**
* **Scikit-learn**
* **Streamlit**
* **TMDB API**

---

## 📊 Machine Learning Approach

### 🔹 Collaborative Filtering

* User-User Similarity
* Cosine Similarity

### 🔹 Matrix Factorization

* Implemented using SGD
* Optimized using RMSE

### 🔹 Final Deployment Model

* Item-based similarity for fast recommendations

---

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/movie-recommender-system.git
cd movie-recommender-system
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
